# Data file description

List of Bacillus phages used during the workshop.

Phage genome sequences were obtained from the Sequence Read Archive (SRA).

- `Frank.fasta` : Sequencing of Bacillus phage Frank (https://trace.ncbi.nlm.nih.gov/Traces/?run=SRR37000638)
- `Hari.fasta` : Sequencing of Bacillus phage Hari (https://trace.ncbi.nlm.nih.gov/Traces/?run=SRR37000635)
- `JackRabbit.fasta` : Sequencing of Bacillus phage JackRabbit (https://trace.ncbi.nlm.nih.gov/Traces/?run=SRR37000633)
